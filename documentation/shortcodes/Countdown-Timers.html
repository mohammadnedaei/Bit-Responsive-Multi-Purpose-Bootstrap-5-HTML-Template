   <p id="days"></p><span> Day(s)</span>
</div>
<div class="hours-timer primary-timer">

</div>
</div>

Make sure to use alerts in proper HTML structure for best experience.

Countdown-Timers Class List :

    Defining Timer :
  • Necessary timer ==> Activates timer (define in parent)
  • Necessary days-timer + #days ==> Defines days
  • Necessary hours-timer + #hours ==> Defines hours
  • Necessary mins-timer + #mins ==> Defines minutes
  • Necessary secs-timer + #secs ==> Defines seconds

How to set the timer?

You can set the timer by doing below steps:

  • Go to js/main.js and head into section 17.
  • You will see a code block defining future like so: future = Date.parse("aug 2, 2021, 11:30:00");
  • Change The time and save.
