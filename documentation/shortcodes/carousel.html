Elements & Shortcodes

Carousel

Bit uses OwlCarousel for it's default slider. You can customize it by doing below steps. See code example & Demo for more information.

Copy 
                                                      
<div class="small-item-carousel black owl-carousel mb-0">
<div class="logo-item">
   <img src="images/brand/logo-1.png" width="700" height="200" alt="logo-1">
</div>
<div class="logo-item">
   <img src="images/brand/logo-2.png" width="700" height="200" alt="logo-2">
</div>
</div>

OwlCarousel Sliders Class List :

  • fullwidth-slider
  • fullwidth-slider-fade
  • fullwidth-gallery
  • item-carousel
  • small-item-carousel
  • single-carousel
  • content-slider
  • photo-slider
  • work-full-slider
  • blog-posts-carousel
  • blog-posts-carousel-alt
  • image-carousel
  • fullwidth-slideshow
  • fullwidth-slideshow-pager


    • Carousel Options

    You can also set different options for each Carousel Slider you want. Just head into js/carousel.js and select your slider.

    You can change options in slider function. Option's value can be true, false, type & number according to the option.

    You can see full list of Carousel options below with default value.


    • itemsTabletSmall: false
    • singleItem: false
    • itemsScaleUp: false
    • slideSpeed: 200
    • paginationSpeed: 800
    • rewindSpeed: 1e3
    • autoPlay: false
    • stopOnHover: false
    • navigation: false
    • navigationText: ["prev", "next"]
    • rewindNav: true
    • pagination: true
    • paginationNumbers: false
    • responsive: true
    • responsiveRefreshRate: 200
    • baseClass: "owl-carousel"
    • theme: "owl-theme"
    • lazyLoad: false
    • lazyFollow: true
    • lazyEffect: "fade"
    • autoHeight: false
    • jsonPath: false
    • jsonSuccess: false
    • dragBeforeAnimFinish: true
    • mouseDrag: true
    • touchDrag: true
    • addClassActive: false
    • transitionStyle: false

    You can use different Carousel animation type by changing its class name. See below animation list for more information:

    • owl-origin
    • owl-fade-out
    • owl-fade-in
    • owl-backSlide-out
    • owl-backSlide-in
    • owl-goDown-out
    • owl-goDown-in
    • owl-fadeUp-in
    • owl-fadeUp-out
