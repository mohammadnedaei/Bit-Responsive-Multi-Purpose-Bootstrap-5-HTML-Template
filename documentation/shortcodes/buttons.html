    Buttons

    See Below Buttons Demo & Class lists for more information.

    Code Example

    Copy 
                                                                   
                                                                        <a href="#" class="material-btn primary-btn lg-btn">Large</a>

    Make sure to use button classes in <a> for best experience.

    Buttons Class List :

    • Necessary material-btn ==> Material button made by BIT (add this class to every material button)
    • btn ==> Bootstrap button
      • Button Sizes :
    • mini-btn ==> Mini material button
    • sm-btn ==> Small material button
    • md-btn ==> Medium material button
    • lg-btn ==> Large material button
      • Button Colors :
    • primary-btn ==> Primary material button
    • secondary-btn ==> Secondary material button
    • info-btn ==> Info material button
    • light-btn ==> Light material button
    • dark-btn ==> Dark material button
      • Button types :
    • rounded-btn ==> Rounded material button
    • rainbow-btn ==> Rainbow button
      • Outline Buttons :
    • primary-outline-btn ==> Primary Outline material button
    • secondary-outline-btn ==> Secondary Outline material button
    • info-outline-btn ==> Info Outline material button
    • dark-outline-btn ==> Dark Outline material button
    • light-outline-btn ==> Light Outline material button
      • Buttons with animated gradient :
    • primary-anim-grad-btn ==> Primary Animated Gradient material button
    • secondary-anim-grad-btn ==> Secondary Animated Gradient material button
    • info-anim-grad-btn ==> Info Animated Gradient material button
    • dark-anim-grad-btn ==> Dark Animated Gradient material button
    • light-anim-grad-btn ==> Light Animated Gradient material button
      • Button Animations :
    • jelly-btn ==> Jelly Effect on Hover material button
    • pulse-btn ==> Light Pulse material button
    • heavy-pulse-btn ==> Heavy Pulse material button
    • tada-btn ==> Tada Effect material button
    • shadow-btn ==> Shadow material button
    • tilt ==> Tilt Effect on Hover material button
      • Icon Buttons :
    • <a class="material-btn"><i class="fas fa-shopping-cart"></i>Text</a> ==> LTR material icon button
    • <a class="material-btn">Text<i class="fas fa-shopping-cart"></i></a> ==> RTL material icon button
    • non-text-btn ==> Material icon button without text
      • Block Buttons :
    • block-btn ==> Block material button
      • Bootstrap 5 Buttons :

      Make sure to read full Bootstrap 5 Buttons documentation before using them.
