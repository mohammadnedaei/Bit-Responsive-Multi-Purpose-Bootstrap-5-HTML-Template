Elements & Shortcodes

Progress bars

See Below Progress bars Demo & Class lists for more information.

Code Example

Copy 
                                                               
 <div class="progress-bar-card col-lg-4 col-md-6 col-sm-12 primary-progress">
 <div class="progress-bar-box wow fadeIn ">
    <div class="progress-bar-percent m-auto">
       <svg class="md-progress">
          <circle cx="70" cy="70" r="70"></circle>
          <circle class="yellow-progress pc-90" cx="70" cy="70" r="70"></circle>
       </svg>
       <div class="progress-bar-number">
          <h2>90</h2>
          <span>%</span>
       </div>
    </div>
    <h2 class="progress-bar-text">Primary</h2>
 </div>
</div>

Make sure to use alerts in proper HTML structure for best experience.

You can change progress bars percent by changing .pc-number.

Also note that you have to use number based on 5 meaning that you can choose pc-5, pc-10, pc-15, ... pc-100 Updates Soon
