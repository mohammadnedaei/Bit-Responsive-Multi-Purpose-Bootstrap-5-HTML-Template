Elements & Shortcodes

Tabs

First of all, You should set your tabs. See below code example and image for more information.

Copy 
                                                               
<!--* Nav tabs Start -->
<ul class="nav tpl-alt-tabs font-alt pt-30 pt-sm-0 pb-30 pb-sm-0 ">
   <li class="m-auto">
      <a href="#first-tab" data-bs-toggle="tab" aria-label="tab-item">
         <div class="alt-tabs-icon mb-0">
            <span class="fas fa-copyright fa-3x wow flipInX tab-icon active mb-0"
               data-wow-duration="2s"></span>
         </div>
         <h4 class="tab-icon-des">BRANDING</h4>
      </a>
   </li>
</ul>
...
...
...
<!--* Tab panes Start -->
<div class="tab-content tpl-tabs-cont">
   <!--* First Tab Item Start -->
   <div class="tab-pane fade in show active" id="first-tab" role="tabpanel">
      <div class="section-text">
         <div class="row">
            <div class="col-md-4 mb-md-40 mb-xs-30">
            ...
            </div>
            ...
         </div>
      </div>
   </div>
   <!--* First Tab Item End -->
</div>

Notice!

Tab Item href and it's description id must be the same.
You can set your active tab using .active in it's description.
