Page Elements

Scrolling:

You can navigate between sections with smooth scrolling. Just create <a href="#destination">Destination</a> and point it's href to your destination section ID. See below code example for more information.

Navigation Code Example

Copy 
                                                        
<nav id="main-nav" class="navbar fixed-top navbar-expand-lg bg-scrolling-dark">
 ...
<div class="collapse navbar-collapse" id="main_nav">
   <ul class="navbar-nav mx-auto">
     ...
      <li class="nav-item single-nav-item home-nav-item">
   -->      <a class="nav-link" href="#home"> HOME </a>
      </li>
      <li class="nav-item single-nav-item about-nav-item">
    -->     <a class="nav-link" href="#about"> ABOUT US </a>
      </li>
        ...
      </li>
   </ul>
</div>
</nav>

Go to Top Code Example

Copy 
                                                      
<div>
    <a id="back-to-top" href="#home" aria-label="back-to-top"><i class="fas fa-chevron-circle-up"></i></a>
</div>
