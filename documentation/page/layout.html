Page Elements

Layout:

BIT includes 5 types of page layouts. Choose among them and make your website different.

  • Full Width
  • Left Sidebar
  • Right Sidebar
  • Bottom Sidebar - Masonry
  • Top Sidebar - Masonry
-Full Width:

See below code Example & image link for more information.

Code Example

Copy 
                                                        
<!-- Main Section Start -->
<section class="page-section" id="main">
  <div class="container relative">
    <div class="row">
       <div class="container">
          <!-- Post Start -->
          <div class="blog-item">
            <!-- Post Title -->
            <h2 class="blog-item-title font-alt"><a class="info-hover-grad" href="#">Media Gallery Post</a>
            </h2>
            <!-- Author, Categories, Comments -->
            <div class="blog-item-data">
                          ...
            </div>
            <!-- Media Gallery -->
            <div class="blog-media">
                          ...
            </div>
            <!-- Text Intro -->
            <div class="blog-item-body">
            <p>
                            ...
            </p>
            </div>
            <!-- Read More Link -->
            <div class="blog-item-foot">
            <a href="#" class="material-btn dark-btn rounded-btn"><i
                              class="fas fa-angle-double-right home-btn-logo"></i>Read More</a>
          </div>
        </div>
      </div>
    </div>
  </div>
</section>
<!-- Post End -->
