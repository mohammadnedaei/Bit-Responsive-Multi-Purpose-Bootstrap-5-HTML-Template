Getting Started

Changing Image Root

Choose one of the below steps for changing image root in your website.

1- Replace your image with the current image in images/ folder.

2- Place your image in images/ folder and then change root manually.

You may need to update root by changing src in <img> or background-image in <section>or<div> as you need.

LOGO & Favicon Settings:

Favicon:

Both Logo & Favicon images are located in images/logo. You can replace your own Icon with these files.

Code example for favicon

Copy 
                                          
<!--! Favicons -->
<link rel="apple-touch-icon" sizes="180x180" href="images/logo/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="images/logo/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="images/logo/favicon-16x16.png">
<link rel="manifest" href="images/logo/site.webmanifest">
LOGO:

You can use your logo in the loader, header, about section and footer.

Code example for LOGO in loader:

Copy 
                                          
<!--* LOGO in loader -->
<div class="page-loader">
   <div class="loader" style="background-image: url('images/logo/logo-light.png')"></div>
    ...
</div>

Code example for LOGO in header:

Copy 
                                          
<!--* LOGO in header -->
<nav id="main-nav" class="navbar fixed-top navbar-expand-lg bg-scrolling-dark">
  <a class="glow logo-text-1 navbar-brand" href="#" aria-label="nav-brand">
     <img width="64" height="64" src="images/logo/favicon-light.png" alt="header-logo">
  </a>
    ...
</nav>

Code example for LOGO in about section:

Copy 
                                          
<!--* LOGO in about section -->
<div class="col-lg-4 col-md-4">
  <img width="256" height="256" class="about-logo" src="images/logo/logo-light.png" alt="logo-light">
</div>

Code example for LOGO in header:

Copy 
                                          
<!--* LOGO in footer -->
<div class="footer-logo-part">
  <img width="256" height="256" id="footer-logo" src="images/logo/logo-light.png"
     alt="footer-logo" class=" logo-footer wow rubberBand">
     ...
</div>
