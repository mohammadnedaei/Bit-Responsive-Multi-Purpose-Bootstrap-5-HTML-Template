Portfolio

Portfolio Setup:

Setting up portfolio filter & items in BIT is very simple. Just follow below steps to do so.

- Code Example for setting up portfolio filter:

Copy 
                                                      
<!--* Item Filters Start -->
<div class="items-filter font-alt align-center">
   <a href="#" class="filter active" data-filter="*" aria-label="bit-btn">All Other
      works</a>
   <a href="#" class="filter mt-xs-10" data-filter=".developing" aria-label="work-btn">web
      Developing</a>
   <a href="#" class="filter mt-xs-10" data-filter=".designing" aria-label="work-btn">Art &amp;
      Graphics</a>
   <a href="#" class="filter mt-xs-10 mt-lg-20" data-filter=".photography"
      aria-label="work-btn">Photo
      works</a>
</div>
<!--* Item Filters End -->

Portfolio Filters & Grid

data-filter="*" points to all portfolio items.

Each portfolio item has it's class name for seperating from others, but you can mix them using .mix in portfolio item.

You can Choose among 4 Types of Portfolio grids in Portfolio Types.

All 4 Portfolio Types come with wide, overlay & lightbox options which you can customize.

See below Portfolio Grid code example for more information:

Copy 
                                                  
<!--* Item Grids Start -->
<ul class="isotope-grid items-grid item-grid-4 clearfix font-alt hide-titles" id="item-grid">

- Code Example for setting up portfolio items:

Copy 
                                                  
<li class="grid-item mix photography">
<a href="images/portfolio/lightbox/38.webp" class="bit-lightbox-link mfp-image"
      aria-label="portfolio-item">

      </div>

         <h2 class="item-title">Tidy</h2>
         <div class="item-descr">
            a Multi-Purpose Website
         </div>
      </div>
   </a>
</li>
