Header

Header Types:

  • Transparent Header

    Background on scrolling

    You can use below class lists to create this header.

    • bg-scrolling-dark
    • bg-scrolling-light
    • bg-scrolling-red
    • bg-scrolling-yellow
    • bg-scrolling-green
    • bg-scrolling-blue

    New Colors & Gradients Coming Soon!

  • Fixed Background

    Fixed Background

    You can use below class lists to create this header.

    • bg-fixed-dark
    • bg-fixed-light
    • bg-fixed-primary
    • bg-fixed-secondary
    • bg-fixed-info
    • bg-fixed-compound

    New Colors & Gradients Coming Soon!

  • Glassy Header

    Glassy Header

    You can use below class lists to create this header.

    • glassy-header-dark
    • glassy-header-light
    • glassy-header-primary
    • glassy-header-secondary
    • glassy-header-info
    • glassy-header-blue
    • glassy-header-green
    • glassy-header-yellow
    • glassy-header-red

    New Colors & Gradients Coming Soon!

    You can use Glass Effect on Your Header using .glass-scrolling class.

    glass scrolling demo

  • Classic Header

    Classic Header

    You can remove .fixed-top from header & add .bg-fixed-color or .glassy-header-color to create this header.

    classic header demo

  • Modern Header

    Modern Header

    You can add .hide-on-scroll to header to create this header.

    We recommend you to use .glassy-header-color for this type of header.

    modern header demo

  • Center Mega Menu

    Center Megamenu

    You can add .megamenu to header to create this header.

    Make sure to check out demo above for best experience.

  • Center Navigation

    Center Navigation

    You can Center your navigation by adding mx-auto to it.

  • Right alignment

    Right align Navigation

    You can align your navigation to right by adding ml-auto to it.

  • Left alignment

    Left align Navigation

    You can align your navigation to Left by adding mr-auto to it.

  • Center Logo

    Make sure to check out demo above for best experience.

  • Dark Topbar

    Make sure to check out demo above for best experience.

  • Light Topbar

    Make sure to check out demo above for best experience.

  • Boxed Header

    Use .boxed-nav to create this type of header.

    Make sure to check out demo above for best experience.

  • Modern Hamburger Menu (Half)

    Use .hamburger-menu-half to create this type of header.

    Make sure to check out demo above for best experience.

  • Classic Hamburger Menu (Full)

    Use .hamburger-menu-full to create this type of header.

    Make sure to check out demo above for best experience.

    Classic Hamburger menu full demo

  • Hamburger Menu With Blur Effect

    Use OpenNavWithBlur() Calling function on hamburger icon to create this type of header.

    Make sure to check out demo above for best experience.
